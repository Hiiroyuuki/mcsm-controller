# MC Server Manager Controller
## Now you can use Openclaw, powered by AI, to operate your server using natural language.

### This project enables Openclaw to understand how to manage a Minecraft server through MCSM. We aim to make Minecraft server operations more efficient and simple.

## Dependenceise:
Openclaw  
MC Server Manager  
Python

## How to install:
Move this file to the skills folder of openclaw.  
Asking the openclaw, she will help you to finish the initialization and setup.
